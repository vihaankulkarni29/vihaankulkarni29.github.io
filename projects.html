WildTypeAligner

2024 Active

An automated batch pairwise alignment tool that processes thousands of protein sequences against a reference wild-type sequence to rapidly identify mutations and variants. Built to handle high-throughput sequence analysis workflows common in AMR surveillance studies.

WildTypeAligner integrates with standard BLAST algorithms while adding custom parsing and reporting features that make mutation identification straightforward. The tool generates detailed reports highlighting mutation positions, types, and frequencies across large sequence datasets.

MetaAcpPipeline

2023-2024 Active

A comprehensive Snakemake-based computational pipeline designed to map protein variation across Indian E. coli isolates, with a focus on identifying regional mutation hotspots in antimicrobial resistance genes. This pipeline combines multiple analysis steps into a reproducible, scalable workflow.

The pipeline handles everything from raw sequence data to final visualization, including quality control, alignment, variant calling, and statistical analysis. It's designed for HPC environments but can scale down to run on standard workstations. Special attention was paid to making the pipeline configurable for different study designs and bacterial species.

FastaAAExtractor

2024 Active

A Python-based automation tool that extracts targeted amino acid sequences from unannotated genomic data. Designed to eliminate the manual curation bottleneck that slows down comparative genomics research, this tool reduces processing time by over 90% while maintaining accuracy.

The tool handles batch processing of FASTA files, supports custom extraction parameters, and includes robust error handling for real-world messy genomic data. It's particularly useful for researchers working with bacterial genomes in antimicrobial resistance studies.
